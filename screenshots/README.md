# Project Results and Screenshots

This folder contains the simulation, schematic, synthesis, and FPGA implementation results of the 32-bit Single Cycle Processor project.

## Contents

- **simulation_waveform.png** – Behavioral simulation waveform showing processor execution.
- **schematic.pdf** – RTL schematic of the processor architecture.
- **synthesis schematic.pdf** – Synthesized design generated by Vivado.
- **implementation_device_view.png** – FPGA device placement view after implementation.
- **implementation_routing_view.png** – Routing view showing FPGA interconnections.
- **32-bit processor architecture diagram.jpeg** – High-level processor block diagram.
